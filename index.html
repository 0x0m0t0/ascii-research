ASCII ART

pronunciation: /ˈæskiː/ ASS-kee

Textual based art made with ASCII characters.

Also loosely used to refer to general computer textual art

ASCII stands for American Standard Code for Information Interchange.

  • A character coding standard for electronic communication.
  • 1st edition was published in 1963
  • 128 characters total, but only 95 printable
  • Made for the english language so no accents or loanwords

  • ASCII standard is a bit deprecated today and replaced by Unicode
  • Unicode, current unifying standard supporting 161 scripts
  • Emoji included ⭐⭐⭐

WHY

- performant
- very low tech
- create interesting visuals with restrictions
- creative coding
- minimal
